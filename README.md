# SmallNet Docs

SmallNet Docs is a Gemini-native documentation hub for the **small web**, with an initial focus on **Gemini** and a primary focus on **newcomers**.

This project aims to be a **best-effort map** of resources, tools, and practical guidance:
- curated listings (annotated links)
- lightweight reference pages
- a few “house guides” for common newcomer/operator tasks

It is **not official** and may contain mistakes. Corrections are welcome.

## Access
SmallNet Docs is intended to be read with a **Gemini client**.
